PRD · Predictive History

Produto: Plataforma de modelagem matemática de contexto global político, social e econômico, com foco em previsão probabilística de cenários.

Multiagente Grafo Multiplex Dinâmico Teoria dos Jogos Camada Étnica Simulação de Cenários

1) Objetivo (BLUF)

Construir um motor que estime trajetórias prováveis (não eventos exatos) para risco de conflito, estabilidade institucional, mudanças de coalizão e resiliência econômica, a partir de dados históricos e estado atual da rede global.

2) Escopo V1

  • Modelo de dados unificado para players, grupos, relações e eventos.
  • Rede em camadas: política, econômica, social/informacional e étnica.
  • Motor de simulação por passos de tempo (t).
  • Cenários com choques exógenos (sanções, energia, eleições, crise).
  • Dashboard de risco e evolução de métricas.

3) Resultado esperado

Saídas principais por janela temporal:

  • Prob. de conflito (interno/externo)
  • Índice de estabilidade institucional
  • Polarização social / saliência étnica
  • Reconfiguração de coalizões
  • Stress macroeconômico

5) Banco de dados (recomendação V1)

PostgreSQL como base única, com extensões:

  • TimescaleDB para séries temporais.
  • pgvector para embeddings de narrativas/notícias.

Adicionar Neo4j só quando análises de grafos em escala forem gargalo.

6) Estruturas mínimas

  • players, groups, memberships
  • relations (layer, weight, signed, directed)
  • events, indicators
  • narratives (texto + embedding)
  • simulation_runs, simulation_steps

7) Dinâmica do motor (cada passo t)

  1. Atualiza estado dos players (recursos, legitimidade, risco, coesão).
  2. Calcula payoff multicritério (poder, segurança, riqueza, identidade).
  3. Escolha de ação por agente (cooperar, competir, neutralizar, coalizar).
  4. Atualiza rede multiplex e pesos das arestas.
  5. Aplica choque exógeno, se houver.
  6. Registra métricas de saída e estado.

8) Critérios de sucesso

  • Backtesting: reproduzir padrões de eventos históricos com erro aceitável.
  • Robustez: sensibilidade controlada a parâmetros críticos.
  • Explicabilidade: cada previsão deve indicar fatores contribuintes.
  • Operacional: geração de cenários em minutos, não horas.

9) Cenário-base para validação

Sanções + choque de energia + eleição polarizada

Saídas alvo: risco de ruptura institucional, conflito interno, mudança de blocos e impacto em inflação/desemprego/apoio político.
