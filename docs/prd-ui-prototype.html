Checklist UX Decisions + Rationale (curto)

✅ Arquitetura em 3 colunas para leitura executiva: contexto (esquerda), análise principal (centro), explicabilidade/alertas (direita).
✅ Tabs por intenção (Simulação, Riscos, Eventos) para reduzir carga cognitiva e organizar decisões por tarefa.
✅ Slider temporal com feedback imediato (T-12→T+12) para reforçar causalidade temporal e facilitar storytelling analítico.
✅ Cenários com estados simulados (baseline/sanções/energia/eleição) para comparação rápida sem backend pronto.
✅ Filtros mínimos viáveis (Região/Horizonte/Layers) para validar utilidade antes de adicionar filtros complexos.
✅ Mobile-first adaptativo: layout colapsa para 1 coluna e preserva legibilidade dos principais KPIs.
